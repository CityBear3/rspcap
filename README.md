# rspcap

### Usage
1. Clone this repo
```
git clone "url"
```
2. move project root directly
```
cd rspacp
```
3. Build
```
cargo build
```
#### Note: You should not use "sudo cargo run".
3. run
```
#localhost
sudo ./target/debug/rspacp lo

#eth0
sudo ./target/debug/rspacp eth0
```
